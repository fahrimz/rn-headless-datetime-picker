# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, a build tool that works with multi-package repos, or single-package repos to help you version and publish your code. You can find the full documentation for it [in our repository](https://github.com/changesets/changesets).

We have a quick list of common questions to get you started engaging with this project in [our documentation](https://github.com/changesets/changesets/blob/main/docs/common-questions.md).

## Workflow for this repo

1. Make a code change on a feature branch.
2. Run `yarn changeset` and answer the prompts — it creates a markdown file in this folder describing the bump (patch / minor / major) and a summary line for the changelog.
3. Commit that file along with your code change. Open a PR.
4. After the PR merges to `main`, the release workflow opens (or updates) a "Version Packages" PR that bumps `package.json` and rewrites `CHANGELOG.md`.
5. Merge the "Version Packages" PR. The release workflow then publishes to npm and creates the GitHub release + tag.
