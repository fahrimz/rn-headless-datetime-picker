# rn-headless-datetime-picker

Headless date and time picker for React Native — promise-based API, Expo-friendly

## Installation


```sh
npm install rn-headless-datetime-picker
```


## Usage


```js
import { multiply } from 'rn-headless-datetime-picker';

// ...

const result = multiply(3, 7);
```
